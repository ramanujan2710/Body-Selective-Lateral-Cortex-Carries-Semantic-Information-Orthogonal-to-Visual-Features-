# Project Overview

This project focuses on analyzing how the body-selective lateral cortex carries semantic information that is orthogonal to visual features.

## Project Structure

- **data/**: Contains all raw and preprocessed data used for analysis.
- **scripts/**: Includes scripts for data processing, visualization, and model training.
- **notebooks/**: Jupyter notebooks for exploratory data analysis and results visualization.
- **results/**: Contains output files generated by the analysis scripts, including figures and processed data.
- **README.md**: This file, containing an overview and documentation of the project.

## Functionality

- **Data Processing**: The scripts in the `scripts/` directory preprocess the raw data for analysis. This includes normalization, feature extraction, and any necessary transformations.
- **Model Training**: The project includes scripts for training machine learning models to understand the relationship between body-selective signals and semantic content.
- **Visualization**: Jupyter notebooks provide visual representations of the data and the outcomes of the analyses, making it easier to understand the results.

## Getting Started

To get started with the project, clone the repository and install the required dependencies listed in the `requirements.txt` file. Use the Jupyter notebooks for step-by-step exploration of the data and analysis results.
